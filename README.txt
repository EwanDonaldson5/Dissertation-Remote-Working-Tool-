As this project is not distributable the following must be setup to run the application;
1. Install the Apache web server onto your machine
2. Setup phpMyAdmin on your web server (allowing for a local database)

The images folder of the application is included seperately from the project folder.

To operate the Dropbox feature a new account must be made for each company.
This accounts access code should be inserted/updated into dropbox.java file.
The link used in the downloading aspect of the feature should also be updated to match
the new Dropbox account.
